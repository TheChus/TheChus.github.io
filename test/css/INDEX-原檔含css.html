Despicable Me

在和諧美麗的郊野社區，家家戶戶前院有鮮豔玫瑰、白色圍籬的裝飾。但唯有一戶人家，不但雜草叢生、烏漆抹黑，其中更藏匿一個不為人知的祕密基地。這裡是做壞事做上癮的世界級大壞蛋格魯，他不但坐擁各種最先進的科技產品，一發起狠來，更會使用縮小光、冷凍光和陸空戰車，來消滅阻止他偷月亮計畫的人。

不料他竟然踢到鐵板= =，三個活潑可愛的孤兒不速而降，搞得他手足無措、兵荒馬亂，眼看他策畫已久、史上最大的一票的偷月計劃即將展開…，這三個寶貝究竟會與格魯擦出什麼樣的火花…，偷月計劃能如願成功嗎？還是格魯會乖乖服貼成一個他想都沒想過超級奶爸呢？
